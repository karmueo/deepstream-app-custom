# C-UAV 光电设备报文协议

## 1 通用协议

### 1.1 传输协议

- **传输方式**: UDP 组播
- **组播地址/端口**: 通过配置文件修改
- **默认地址**: `230.1.88.51:8003`（指控中心 → 光电）
- **默认反馈地址**: `230.1.88.51:8013`（光电 → 指控中心）

### 1.2 报文格式

报文采用 JSON 格式，分为两部分：**公共内容** + **具体信息**

#### 1.2.1 单信息格式

```json
{
  "公共内容": {},
  "具体信息": {}
}
```

#### 1.2.2 多信息格式

```json
{
  "公共内容": {},
  "cont": [
    { "具体信息1": {} },
    { "具体信息n": {} }
  ]
}
```

### 1.3 注意事项

1. 组播地址和端口通过配置文件修改
2. 站址和设备 ID 通过组播装订
3. BIT 状态周期性自动上报
4. 中心下发控制到设备，设备收到控制后反馈

### 1.4 默认值

默认设备未配置时：
- `sys_id = 999`
- `dev_type = 999`
- `dev_id = 999`
- `subdev_id = 999`
- 向 `0/0/0/0` 默认地址回馈信息

## 3 查询指令

### 3.1 查询单设备

```json
{
  "cmd_id": 0,
  "cmd_coef1": 0
}
```

### 3.2 查询所有设备

#### 3.2.1 指定类型下所有设备

- `rx_dev_type = x`（指定设备类型）
- `rx_dev_id = 999`（不指定设备）

#### 3.2.2 所有类型下所有设备

- `rx_dev_type = 999`
- `rx_dev_id = 999`

## 8 光电跟踪控制

**报文ID**: `0x7203` (29187)

格式：单信息

| 字段 | 类型 | 说明 |
|------|------|------|
| `trk_end` | int | 跟踪模块：`0`退出 `1`打开，默认1 |
| `pt_trk_link` | int | 光电联动：`0`停止 `1`开始，`255`无效 |
| `ir_trk_link` | int | 红外联动：`0`停止 `1`开始，`255`无效 |
| `trk_str` | int | 跟踪开关：`0`停止 `1`开始，`255`无效，默认1 |
| `trk_dev` | int | 跟踪设备：`0`可见光 `1`红外 `3`多传感器联动，默认3 |
| `trk_mod` | int | 跟踪模式：`0`自动 `1`半自动 `2`手动，默认0 |
| `det_trk` | int | 检测跟踪：`0`检测 `1`识别，默认1 |

**备注**：
- `trk_mod = 0` 时，`det_trk` 只能发 1
- `trk_mod = 1/2` 时，`det_trk` 只能发 0
- 切换目标时：停止联动 → 停止跟踪 → 引入新目标 → 开始联动 → 开始跟踪
